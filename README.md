# Netflix-Data-Cleaning-Project
Performed end-to-end data cleaning on Netflix Movies and TV Shows dataset for analysis readiness.

This project focuses on cleaning and preprocessing the Netflix Movies and TV Shows dataset.

## Tasks Performed
- Imported dataset using Pandas and NumPy
- Checked dataset shape and structure
- Identified missing values
- Filled missing values using fillna()
- Checked duplicate rows
- Removed duplicate values
- Standardized text formatting
- Converted date column into proper format
- Renamed columns for consistency
- Exported cleaned dataset

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Files Included
- netflix_titles.csv (Original dataset)
- cleaned_netflix_dataset.csv (Cleaned dataset)
- Netflix Dataset.ipynb (Python code notebook)
